# PopiiMyPlax Gaming Hub

Landing publica del canal PopiiMyPlax.

## Publicacion

La web se publica desde la carpeta `public/` mediante GitHub Pages.

El repositorio puede permanecer privado. La web publicada sera publica para visitantes, pero no editable por ellos.

## Seguridad

- No subir contrasenas, tokens, claves API ni archivos `.env`.
- No anadir colaboradores si solo el propietario debe modificar la web.
- Mantener GitHub Pages configurado con GitHub Actions y despliegue desde `public/`.
